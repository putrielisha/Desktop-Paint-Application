# Desktop-Paint-Application
A C# desktop paint application featuring a simple user interface for freehand drawing and basic editing tools.

## Overview
Paint Studio is a feature-rich desktop painting application built using C# and Windows Forms (.NET). It goes beyond a basic paint tool by including multiple drawing modes, image editing features, and a custom splash screen experience with audio and UI transitions.

## Key Features
### Application Startup
- Custom splash screen (“Paint Studio” launch screen)
- Logo display with background music
- Launch / Exit navigation system

### Drawing Tools
- Freehand brush tool
- Adjustable brush size
- Eraser tool with size control
- Paint bucket (fill tool)
- Shape tools (line, circle, rectangle)
- Text tool with customizable font and size
- Emoji brush support

### Editing Features
- Color scale for drawing tools
- Change UI background theme (not canvas)
- Greyscale / color filter effects
- Clear canvas option
- Import images / coloring pages

### File Features
- Save artwork as image file
- Load external images into canvas

## Built with
- C#
- .NET Framework (Windows Forms)
- Visual Studio
